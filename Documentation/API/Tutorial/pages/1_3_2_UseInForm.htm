RowManager - Use In A Form

Now take your spiffy new RowManager, and use it in a form.

Let's create a form to enter a new series into your series table. The HTML can be very basic for this example:

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
<html>
<head>
<meta http-equiv="Content-Type" content="text/html; charset=iso-8859-1">
<title>FormExample</title>
</head>
<body>
<h1>Enter A new Series:</h1>
<form action="" method="post">
Name: <input name="title" type="text"><br>
<input type="submit" name="Submit" value="Submit">
</form>
</body>
</html>

Now, we need to add some php code to the beginning to process the data sent by the form. First we check to see if the form has been submitted with form data to be processed.

<?PHP 
// check to see if this form was just submitted:
// if so, there should be the value of the Submit button in the $_REQUEST array
if (isset( $_REQUEST[ 'Submit' ] ) ) {

Once we know there is a form to submit, then we load up the proper include files to make sure our new RowManager will run:

	// load the site objects
	$pathFile = 'General/gen_Includes.php';
	$extension = '';
			
	// Attempt to find proper directory from current page to Root ...
	$numAttempts = 0;
	while ( (!file_exists($extension.$pathFile)) && ( $numAttempts < 5) ) {
		
		$extension = '../'.$extension;
		$numAttempts++; 
	}
		    
	require ( $extension.$pathFile );
	
	/*
	 * This script expects that these files are in the same directory 
	 */
	require( 'SeriesManager.php' );

Then we actually create a new SeriesManager, and store the submitted Series Title into the DB:

	$series = new RowManager_SeriesManager();
	$series->setTitle( $_REQUEST[ 'Title' ] );
	$series->createNewEntry();
}
?>

Copy the included example file to your testing directory and run it from your browser.
