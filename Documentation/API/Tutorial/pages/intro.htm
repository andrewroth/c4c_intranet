Intro

Welcome to the API tutorial. Here we will attempt to provide an easy to use method of becoming accustomed to the programming tools we have developed for building our web applications. This tutorial will step you through a simple process of designing a basic web application from scratch. In the process we will slowly introduce tools we use to handle the operations of operating a web page. As you progress through the tutorial, don't be surprised if you will be asked to rewrite a page you develped previously, using a new tool we are introducing. By the end, your new application will be a fully functional module for our existing framework.
