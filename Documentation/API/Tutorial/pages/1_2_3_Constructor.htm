RowManager Anatomy - Constructor

Now lets take a look at the constructor of your RowManager:

	//CLASS CONSTRUCTOR
	//************************************************************************
	/**
	 * function __construct
	 * <pre>
	 * Initialize this object.
	 * </pre>
	 * @param $initID [INTEGER] The unique id of the episode we are managing.
	 * @return [void]
	 */
    function __construct( $initID=-1 ) 
    {
    
        $dbTableName = RowManager_EpisodeManager::DB_TABLE;
        $fieldList = RowManager_EpisodeManager::FIELD_LIST;
        $primaryKeyField = RowManager_EpisodeManager::PRIMARYKEY_FIELD;
        $primaryKeyValue = $initID;
        
        if (( $initID != -1 ) && ( $initID != '' )) {
        
            $condition = $primaryKeyField . '=' . $primaryKeyValue;
            
        } else {
            $condition = '';
        }
        $xmlNodeName = RowManager_EpisodeManager::XML_NODE_NAME;
        parent::__construct( $dbTableName, $fieldList, $primaryKeyField, $primaryKeyValue, $condition, $xmlNodeName);
        
        $this->dbDescription = RowManager_EpisodeManager::DB_TABLE_DESCRIPTION;
        
        if ($this->isLoaded() == false) {
        
            // uncomment this line if you want the Manager to automatically 
            // create a new entry if the given info doesn't exist.
            // $this->createNewEntry();
        }
    }

For the most part, the code in the constructor is generic and doesn't change for different RowManagers (there are a few exceptions).

 

Init Value:

	function __construct( $initID=-1 ) 
    {

A single initialization value is provided to indicate the value of the primary key you want to automatically load into your RowManager. It that value is not provided, then the RowManager is empty.

 

Auto Creating:

        if ($this->isLoaded() == false) {
        
            // uncomment this line if you want the Manager to automatically 
            // create a new entry if the given info doesn't exist.
            // $this->createNewEntry();
        }

The final if() statement in the constructor is used for those instances when it is desirable for a RowManager to automatically create an entry in the DB. If that is not appropriate for your application, you can remove this portion.
