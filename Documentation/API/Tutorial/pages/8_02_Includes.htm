Modules - Include File

One more thing. In your previous example, you had a controller script that drove your application. At the beginning of that controller script, you included all the files needed by your module.

Now we are going to put all the includes into a single file. The name of the file is incl_[ModuleName].php where [ModuleName] = the name of your application ( in our example it would be incl_TVAddicts.php ).

What To Do

Take the attached example include file and use it to creat the include file for your module.

Notice how the included objects are broken into the different sections that match up with your file layout. Try to keep them organized in this fashion. And make sure you replace the [ModuleName] tag with the name of your Module.

Save this file in the top level of your module directory along with your app_[ModuleName].php file.
