RowManager - Update An Entry In A Table

RowManagers are also used to update the content of a row of data in a table. Continue with the ex_RowManager.php example for the following example.

For this example, we also want to demonstrate how to create a new RowManager that already manages a row of the table. We begin by getting the value of the primary key of the row of data we want to modify, and passing that in as the parameter to the constructor:

// get the ID of the entry we just created
$viewerID = $viewer->getID();

// create a new ViewerManager with the viewerID as the primarykey of the row we
// want to work with.
$newViewer = new RowManager_ViewerManager( $viewerID );

This newViewer object is now managing the row of the table we just created. So now we set the value of the language_id field, and update the table:

$newViewer->setLanguageID( 1 );

$newViewer->updateDBTable();

Now the data in the table will look like:

viewer_id accountgroup_id language_id viewer_userID viewer_passWord viewer_isActive viewer_lastLogin
1 0 1 testUser adkfiownevkdfnvo9sdek 1 0000-00-00

 

Note:

Technically, since the $viewer object created the row in the table, it could easily have done the update operation as well. This example code could have easily been reduced to: 

$viewer->setLanguageID( 1 );

$viewer->updateDBTable();
