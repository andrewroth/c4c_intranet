Templates - Now You Do It

Now it is your turn. Take the web forms you created in a previous example when you create the List of Episodes , and the entry form for the Episodes, and convert them to use Templates for generating their HTML.

In the process we also want to add some error checking to the form. Assume no item is to be left blank. If they did leave an item blank, then display under the form entry box an error message (and don't create a new entry in the DB).
