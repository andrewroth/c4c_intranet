List Iterators - Now You Do It

Up to now in the tutorial, I have presented to you 2 example files that creates a web form using the SeriesManager object, and a web list using a ListIterator from the SeriesManager object.

Now it is your turn.

Create a web form for entering in the specific Episode information, and a web list for displaying a list of each of the episodes in your table.

Note:

Since the episode entries will require being linked to a Season, just hard code a value of 1 for that value right now. The point of this exercise is for you to implement your own form and list using your EpisodeManager object.
