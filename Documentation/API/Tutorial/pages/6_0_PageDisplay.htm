PageDisplay Objects

We are now going to introduce another central object of our Framework: the PageDisplay object.

Just like a RowManager object is responsible for managing the interactions of a database table, the PageDisplay object is responsible for managing the operation of a web page on our system.

Each web page, will have it's own PageDisplay object. The PageDisplay Object provides a basic structure for your page to load its data, process its data, and generate the HTML output for the browser.

In the next few exercises, we will slowly transform your scripts that we have been writing, into full fledged PageDisplay objects.
