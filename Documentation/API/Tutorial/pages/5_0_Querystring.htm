Querystring

In the process of designing your applications, there will be times when one web page will need to direct you to another script, and pass that script some information.

The querystring is the collection of variables passed to your script by adding them to the end of your link. For example:

http://localhost/php5/test/series.php?series_id=3&series_title=BattleStarGalactica

In this above example there are 2 passed in values: series_id, and series_title;

You can retrieve these values in your script by reading them from $_REQUEST['series_id'] and $_REQUEST['series_title'].

To show you how we use querystrings, we will modify our two earlier example scripts to have the Series display list add links to edit an individual Series.

Copy the attached files to your Testing directory. There are 2 new pages here, a Series Display list, and a Series Edit form.

The Series Display List adds on to our previous example by adding an additional Edit Link next to each entry in the table. Clicking on the link will send you to the SeriesEdit page. the Series Edit page will know which entry to edit by the querystring parameter seriesID.

 

Notes:

The form action on the Edit form, needs to keep the querystring values that was passed into the script. If those are not kept, the Edit form will not know which Series entry to update when the form is processed.
