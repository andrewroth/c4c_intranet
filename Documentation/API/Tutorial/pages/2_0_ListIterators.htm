List Iterators

In a previous example, we showed you how to use a ListIterator to select and parse through each row in a table.

In this example, we will build on our previous example to create a web page that displayes each row of data in an HTML table.

(use included example for this part)

At the point that the table is displayed, we create a new SeriesManager, request a ListIterator containing all the row in the table, and then echo out a new HTML row with the title of the series as the content:

<body>
 <p>Current TV Addictions:</p>
 <table width="100" border="0" cellspacing="2" cellpadding="2">
     <tr>
         <td bgcolor="#666666"><span class="text_white">Series</span></td>
     </tr>
<?php

    // Now create a new series manager
    $seriesManager = new RowManager_SeriesManager();
    
    // get all the rows in this table
    $list = $seriesManager->getListIterator();
    
    // for each row
    $list->setFirst();
    while( $series = $list->getNext() ) {
    
        // echo out a new table row with title as contents
        echo '<tr>
         <td>' . $series->getTitle() . '</td>
     </tr>';
     
    } // next row
    
?>
 </table>
</body>
