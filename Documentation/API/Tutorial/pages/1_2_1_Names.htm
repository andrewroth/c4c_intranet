RowManager Anatomy - Names

Lets start by looking at how our new RowManager is declared:

/**
 * class RowManager_EpisodeManager
 * <pre> 
 * This manager keeps track of our favorite tv episodes.
 * </pre>
 * @author Johnny Hausman
 */
class  RowManager_EpisodeManager extends RowManager {

As a common practice, we name our new RowManager in the format: RowManager_[tableName].
(you will also see that the word "Manager" is tacked on to many of our RowManagers, that is an optional keyword that you can include if you feel it makes more since)

And of course, your RowManager needs to extend the RowManager class.

Note:

The format of our comments follow the phpDocumentor guidelines.
