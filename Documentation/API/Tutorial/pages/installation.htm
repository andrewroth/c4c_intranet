Installation

In order for you to run the examples in this tutorial properly, they will have to be run from within the site framework.

What To Do

Before you can begin running these scripts locally, you need to have a web server and php installed on your machine.

You should have already unpacked the site into a folder in order to have access this tutorial. I'll refer to this folder as the "Site" folder.

Now create a testing folder in your Site folder. Use this location to store all the example files included in the tutorial.

To run these files on OS X:

  1. enable web sharing (system preferences->sharing->personal web sharing)
  2. open a web browser
  3. type in the address as: http://localhost/~account/site/testing/filename.php

(note: account is the name of your login account on your computer. if your login userID is "webguru" then you would type in: http://localhost/~webguru/site/testing/filename.php )
