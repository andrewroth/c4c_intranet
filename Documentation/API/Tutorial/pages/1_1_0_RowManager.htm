RowManager

The RowManager is responsible for interacting with a single database table. Any changes to the information in a db table is performed through the RowManager.

RowManagers are used to:

  1. Select a row from a table
  2. Create a new entry in a table
  3. Update an existing entry in a table
  4. Delete a row from a table

The primary role of a RowManager is to work with an individual row of a table.

The next few tutorial pages will introduce you to how row managers work using some existing row managers included in the site framework.
